# StudoTask

## Overview

StudoTask is a student productivity web application designed to help students organize academic activities, manage assignments, monitor deadlines, and visualize schedules through an integrated calendar system.

## Features

### Authentication

* Sign In
* Log In
* LocalStorage-based user session

### Dashboard

* Central navigation page
* Quick access to all features

### Task Manager

* View task list
* Sort by deadline
* Sort by urgency
* Task monitoring

### To-Do List

* Add tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Store tasks in LocalStorage

### Calendar

* Monthly calendar view
* Deadline highlighting
* Task details by date

### Statistics

* Total tasks
* Pending tasks
* Completed tasks
* Progress tracking

## Technology

* HTML5
* CSS3
* JavaScript
* LocalStorage

## Color Palette

Primary: #796042

Background: #FFFFFF

## Running the Project

1. Open project folder in VS Code.
2. Install Live Server extension.
3. Open index.html.
4. Run with Live Server.
